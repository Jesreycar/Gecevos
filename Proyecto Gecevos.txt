1 Descripción de la Aplicación:
La principal función de Gecevos es la administración de los datos del centro educativo,
a nivel de alumnado y profesorado.


Es una aplicación de escritorio.


2 Tecnologías Utilizadas:
* Lucidchart
* Eclipse
* Firebase (Base de datos)


2.1 Lucidchart:
Se trata de un diseñador de interfaces gráficas online. Hemos decidido utilizar esta tecnología por la facilidad de uso, la gran cantidad de posibilidades que ofrece y por la posibilidad de usar esta herramienta de forma gratuita


2.2 Eclipse:
Utilizamos este IDE debido a su fácil uso y compilación en tiempo real. El uso de sus plugins nos permite interactuar de forma gráfica con la interfaz de la aplicación mediante Java Swing.
2.3 Firebase (Base de datos):
Es un servicio de google que permite tener un almacenamiento online y una base de datos en tiempo real.
El motivo de esta elección es por la posibilidad de usarlo de forma gratuita y por el acceso desde cualquier lugar a la base de datos.










3 Pautas de Usabilidad:
3.1 Color (Paleta)
Hemos utilizado colores complementarios y lo hemos combinado con diferentes tonos, sombras, tintes y triádico a partir de un color principal.
3.2 Fuente
Los textos son breves. La gramática es estándar, con un lenguaje claro y coherente.
La fuente usada es Liberation Sans.
3.3 Distribución
Hemos distribuido las diferentes ventanas de una consistente, flexible y legible.
De esta forma nuestra aplicación es fácil de usar y rápida de aprender
